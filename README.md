# Poe AI Wrapper

Simple frontend wrapper for free AI models via Poe-like interface.

## Features
- Clean chat UI
- API endpoint ready
- Easy deploy

## Stack
- HTML/CSS/JS
- Node.js + Express

## Setup
```bash
npm install
node server.js
```

## Roadmap
- Add multi-model support
- Add API key input
